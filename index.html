Your journey #!s here

Shebang is an Artix GNU/Linux based distribution offering a great blend of end user and power user experience.

Using the nimble Openbox window manager, it is highly customisable and provides a modern, full-featured GNU/Linux system without sacrificing performance.

#!/usr/simplicity

From the Openbox website:

Openbox is a highly configurable window manager. It allows you to change almost every aspect of how you interact with your desktop and invent completely new ways to use and control it. It can be like a video game for controlling windows. But Openbox can also be kept extremely simple, as it is in the default setup, meaning that it can suit just about anybody. Openbox gives you control without making you do everything.

Openbox is lightweight and speedy; and as a result, Shebang is fast.

Welcome script at startup: Auto-configures with latest configs, packages and memory-based runtime environments every boot. Simple, clean and repeatable.

Flatpak support: Easily install and sandbox modern applications with Flatpak — wide software availability with better isolation.

You can adjust features like brightness, Bluetooth, Wi-Fi, notification, airplane mode and system upgrade on polybar.

#!/usr/privacy

Shebang has opt-out news widget that notify you privacy tips once in a while.

Ephemeral sessions: Since most of the system lives in RAM, every boot is a fresh environment.

MAC address anonymization: A new MAC address is generated with every connection, prevent device tracking.

Privacy Toggle: Instant control over microphone and camera (eye icon) — hardware-level awareness, no guesswork.

#!/usr/security

Shebang uses runit as the init system; because PID1 must be simple, secure and stable.

XLibre: Fork of the X.org display server, aimed at modernizing X11 and improving security.

Mullvad Browser: The privacy-first browser that's hardened and DNS-isolated. Paired with Flatpak, it ensures your browsing stays private and contained.

Hardened /etc/hosts file: Blocks countless intrusive web services at system level. Less noise, fewer trackers, better privacy.

Encrypted communication: With Quad9 DNS and Tor Routing (lock icon), avoid from leaks and surveillance.

LUKS disk encryption: Protect your full disk with strong encryption, keeping your data unreadable without your passphrase.

System hardening: Secure CPU, GRUB and system-wide configurations are applied automatically for improved protection and performance.
